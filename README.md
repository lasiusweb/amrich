# amrich

Astro website for aquaculture and agriculture product pages. Deployed on [Netlify](https://www.netlify.com/).

## Getting Started

```sh
npm install
npm run dev
```

## Project Structure

```
src/pages/products/    # Product category pages
reports/               # Lighthouse and design critique reports
astro/                 # Astro skill definitions
context/               # Context skill definitions
fullstack/             # Fullstack skill definitions
github/                # GitHub skill definitions
prompt/                # Prompt skill definitions
```

## Deployment

Automatic deployment to Netlify on push to `main`.
